# Dynamic-Mystery-Portfolio
Mystery Portfolio is an interactive developer portfolio built as a mystery-solving experience. Instead of a traditional layout, visitors explore clues, solve puzzles, and interact with dynamic UI elements to discover projects, skills, and achievements. 

# 🕵️ Mystery Portfolio

> **A portfolio where curiosity is the key.**

Forget scrolling through another portfolio website.

In **Mystery Portfolio**, every room hides something about me—but nothing opens for free. Explore a mysterious 3D world, answer simple questions, unlock doors, and gradually discover my projects, skills, experience, and achievements.

Built to make portfolios memorable instead of predictable.

## 🎮 The Experience

You're dropped into a mysterious environment with multiple locked doors.

Each door guards a different part of my portfolio.

To proceed, you'll need to answer a few fun and simple questions. Get them right, and the next room opens. Every unlocked section reveals more about my work as a developer.

No lengthy forms.
No boring navigation.
Just explore, solve, and discover.

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/your-username/mystery-portfolio.git
```

Navigate to the project folder:

```bash
cd mystery-portfolio
```

Open `index.html` in your browser or serve it using a local development server.

---

## 🎲 Gameplay

1. Enter the mystery world.
2. Walk around and inspect locked doors.
3. Answer the challenge presented.
4. Unlock new rooms.
5. Discover hidden sections of the portfolio.
6. Reach the final room and uncover the complete story.

---

## 💡 Why This Project?

Traditional portfolios are meant to be read.

This one is meant to be explored.

The goal was to blend storytelling, interaction, and web development into an experience that keeps visitors engaged while showcasing technical skills in HTML, CSS, JavaScript, and Three.js.
